Lead Your Way Home!

Home Pricing is a free mobile application whose main objective is to help people find the most suitable properties for them in the real estate market. This web app provides not only an user-friendly interface but also a wide range of functionalities that will help satisfy the costumer's needs.

This project was proposed to my group by Deloitte, a widely renowned multinational enterprise in the context of Project Management Laboratory @ FEUP.

Area: Android Development

Languages & Tools: Javascript React Native Redux Node.js Express.js MongoDB Scrapy Python Docker Git
