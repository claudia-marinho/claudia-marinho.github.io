Lead Your Way Home!

Home Pricing is a free mobile application whose main objective is to help people find the most suitable properties for them in the real estate market. This mobile app provides not only a user-friendly interface but also a wide range of functionalities that will help satisfy the costumer's needs when it comes to finding the perfect home.

Lastly, it's important to mention that this project was proposed to my group by Deloitte, a widely renowned multinational enterprise.

Area: Android Development

Course unit: LGP

Languages & Tools: Javascript React Native Redux Node.js Express.js MongoDB Scrapy Python Docker Git
